## Introduction

**Books** is a modern, open source **accounting system** designed to help businesses, freelancers, and small teams manage their finances with clarity and transparency.
Built with **simplicity, scalability, and affordability** in mind.

## Tech Stack

- [Next.js](https://nextjs.org/) – Framework
- [TypeScript](https://www.typescriptlang.org/) – Language
- [Tailwind](https://tailwindcss.com/) – CSS
- [Drizzle](https://orm.drizzle.team/) – ORM
- [PostgreSQL](https://www.postgresql.org/) – Database
- [Better Auth](https://www.better-auth.com/) – Authentication

## Contributing

We love our contributors! Here's how you can contribute:

- [Open an issue](https://github.com/heymasiyo/books/issues) if you believe you've encountered a bug.
- Follow the [local development guide](#) to get your local dev environment set up.
- Make a [pull request](https://github.com/heymasiyo/books/pull) to add new features/make quality-of-life improvements/fix bugs.
